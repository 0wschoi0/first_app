## 📲 실행 방법

Flutter를 먼저 설치해야 합니다. [설치하기](https://docs.flutter.dev/get-started/install)

### 1. Git Clone
```shell
git clone https://github.com/0wschoi0/first_app.git
```

### 2. Flutter 의존성 설치
```shell
flutter pub get
flutter pub upgrade
```

### 3. 실행
```shell
# debug mode
flutter run

# release mode
flutter run --release
```
